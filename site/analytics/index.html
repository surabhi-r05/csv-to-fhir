    üìä Analytics ‚Äî Methods & Outputs¶

    This document summarizes how each analysis was performed, what files were generated, and where to find the results.
    All analyses were run on already-cleaned CSVs located in out/analytics/.

    ‚öôÔ∏è Overview¶

    Primary inputs - patients.csv - encounters.csv - observations.csv

    Key summary outputs - metrics_overview.csv - demographics.csv - los_summary.csv - vitals_summary.csv - cohort_summary.csv - stats_tests.csv

    Charts folder: out/analytics/*.png

    üßæ Metrics Overview (01_load.py, 02_missingness.py, 03_row_counts.py)¶

    Purpose: Assess data completeness and record counts across resources.
    Outputs: - metrics_overview.csv ‚Äî rows & missingness per dataset - missingness_bar.png ‚Äî visual of null percentages - row_counts.png ‚Äî record counts per resource

    Highlights - patients.csv has the highest missingness (~15.9%)
    - observations.csv well-covered (only 1.95% missing)

    üë§ Demographics (04_age_distribution.py, 05_gender_distribution.py, 06_demographics_summary.py)¶

    Methods - AGE = floor((TODAY ‚Äì BIRTHDATE)/365) - Age groups:
    - Child (0‚Äì12) ‚Ä¢ Teen (13‚Äì18) ‚Ä¢ Young Adult (19‚Äì35)
    - Adult (36‚Äì50) ‚Ä¢ Mid-Age (51‚Äì65) ‚Ä¢ Senior (65+)

    Outputs - demographics.csv - Charts: age_distribution.png, age_boxplot.png, age_groups.png, gender_share.png

    Findings - Avg age 49.7 yrs, median 49.0 - Seniors (65+) form 29% of patients (341/1171) - Gender near-balanced: 609 F / 562 M

    üó∫Ô∏è Geography (07_geo_citybubbles.py)¶

    What: City-level bubble map of patient residence.
    Input: CITY, LAT, LON from patients.csv
    Output: patients_city_bubbles.png
    Use: Identify geographic clusters for outreach or service allocation.

    üè• Encounters & LOS (08_encounters_analysis.py)¶

    Computed - LOS_days = (STOP ‚àí START)/86400, clipped ‚â• 0
    - LOS_hours = LOS_days √ó 24 - Outlier removal rule: values > max(p95 per class, 14 days)

    Outputs - los_distribution.csv (cleaned with Id + Patient) - los_summary.csv (count, mean, median, IQR, thresholds) - encounters_by_day.csv - Charts:
    encounters_trend.png ‚Ä¢ median_los_by_class.png
    median_los_by_class_hours.png ‚Ä¢ los_box.png ‚Ä¢ avg_los_by_class.png

    Outlier Thresholds (95th Percentile)¶

    Ambulatory: 0.14 days | Emergency: 0.10 days | Inpatient: 1.17 days
    Outpatient: 0.04 days | Urgent Care: 0.01 days | Wellness: 0.02 days

    ‚ù§Ô∏è‚Äçü©π Observations / Vitals (09_observations_analysis.py)¶

    Steps - Filter numeric observations
    - Map standard LOINC codes ‚Üí vital names
    - Keep latest per patient per vital
    - Compute mean, median, std, min, max, p10, p90, count, coverage %

    Outputs - observation_summary.csv ‚Äî all vitals
    - vitals_summary.csv ‚Äî key vitals (Weight, BMI, BP) - Charts:
    vitals_summary_bar.png ‚Ä¢ bp_vs_bmi_scatter.png ‚Ä¢ vitals_correlation_heatmap.png

    üìä Statistical Tests (11_stats_test.py)¶

    Tests - t-test: LOS (Older vs Others) ‚Üí Welch‚Äôs unequal-variance - œá¬≤ test: Hypertensive-like √ó Gender independence

    Results

    Test Statistic p value Interpretation
    t-test (LOS Older vs Others) ‚àí2.60 0.0093 Significant difference
    œá¬≤ (Hypertensive vs Gender) 0.00 1.000 Independent

    Interpretation - Older adults stay longer on average, significant at p < 0.01.
    - Hypertension-like pattern shows no gender bias.
